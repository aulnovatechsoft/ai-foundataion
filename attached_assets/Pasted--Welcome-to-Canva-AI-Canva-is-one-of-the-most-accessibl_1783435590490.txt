
Welcome to Canva AI
Canva is one of the most accessible tools for creating visual content. And with AI built directly in, going from idea to finished design is significantly faster, no design background required.

In this guide, you'll learn how to unlock Canva AI's full potential, from personal projects to professional work. Because with AI, you're no longer limited to static designs — you can generate text, video, and even code, all from a single prompt.

What Is Canva AI?
Canva AI is a prompt-driven creative tool built directly into Canva. Type an idea, upload a reference image, or speak a request — and it generates a finished, editable design.

Canva AI works across text, images, code, presentations, and video, and it's all in one place.

Here's what the interface looks like.
The Interface
The Canva AI interface has three main areas: a central prompt field at the top, output tabs for switching between Design, Image, Doc, Code, and Video modes, and a sidebar on the left for creating, accessing, and re-editing previous generations.

Here's what each tab does:
Design generates full layouts.
Image produces standalone visuals.
Doc drafts ready-to-use text content.
Code builds interactive elements.
Video creates motion clips.
The prompt field stays the same across all of them — the tab you select determines what comes out.

There is one more way to make changes and adjustments across all tabs. Canva AI accepts three types of input: typed text, uploaded images, and voice.

On desktop or mobile, you can describe what you need, show a reference, or speak the request — whichever feels most natural in the moment.

This matters because you can use each one depending on your task and situation:
A typed description lets you prompt or refine a generation with precise wording.
An uploaded image produces a visual reference to work from.
A voice input produces a prompt through audio for use on the go.
